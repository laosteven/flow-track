## For Coaches

### Viewing a Shared Session (2 easy steps)

1. **Open the link**
   - Click the `flowtrack://` link you received
   - Or open the attached JSON file

2. **Review the session**
   - App opens automatically
   - Tap "View Session" in the dialog
   - All metrics and graphs are ready!

## Requirements

- **Both paddler and coach need the Flow Track app installed**
- **Internet is NOT required** - sharing works offline
- **Supports:** WhatsApp, Email, Telegram, SMS, and any messaging app

## Troubleshooting

### Link doesn't open the app?
- Make sure Flow Track app is installed
- Try tapping the link again
- If that fails, download the JSON file attachment and open it

### Can't share long sessions?
- For very long sessions (30+ minutes), the JSON file attachment is your best option
- The deep link has size limits on some platforms

### Want to keep sessions private?
- Only share links with people you trust
- Shared data includes all session details
- Delete old shared messages if needed
